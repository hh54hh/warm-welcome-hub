# نظام مركز البدر - Badr Center POS System

نظام إدارة نقاط البيع المتطور لمركز البدر، يعمل أونلاين وأوفلاين مع مزامنة تلقائية.

## المميزات

- ✅ **يعمل أونلاين وأوفلاين**: يعمل بدون إنترنت ويتزامن تلقائياً عند الاتصال
- ✅ **واجهة حديثة**: مبنية بـ React + TypeScript + Tailwind CSS
- ✅ **قاعدة بيانات محلية**: IndexedDB مع Dexie للأداء العالي
- ✅ **تطبيق قابل للتثبيت**: PWA يعمل على جميع المتصفحات والأجهزة
- ✅ **إدارة شاملة**: منتجات، زبائن، فواتير، مرتجعات، مخزون
- ✅ **تقارير وإحصائيات**: لوحة تحكم تفاعلية
- ✅ **طباعة فواتير**: دعم الطباعة المباشرة

## متطلبات التشغيل

- Node.js 18+
- npm أو bun

## التثبيت والإعداد

### 1. تحميل المشروع

```bash
git clone <repository-url>
cd github-import-helper
npm install
```

### 2. إعداد قاعدة البيانات السحابية (اختياري)

إذا كنت تريد المزامنة مع Supabase:

1. أنشئ مشروع جديد على [Supabase](https://supabase.com)
2. نفذ الـ SQL schema المرفق في ملف `schema.sql`
3. انسخ URL المشروع و API Key
4. أنشئ ملف `.env` في مجلد المشروع:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

### 3. تشغيل التطبيق

```bash
# للتطوير
npm run dev

# لبناء الإنتاج
npm run build

# لتشغيل الإنتاج محلياً
npm run preview
```

## هيكل قاعدة البيانات

### الجداول المحلية (IndexedDB)

- `products`: المنتجات والأسعار والمخزون
- `categories`: فئات المنتجات
- `customers`: بيانات الزبائن
- `invoices`: الفواتير والمبيعات
- `returns`: المرتجعات
- `payments`: المدفوعات
- `movements`: حركات المخزون
- `settings`: إعدادات النظام

### الجداول السحابية (Supabase)

- `products`: المنتجات
- `customers`: الزبائن
- `sales`: المبيعات
- `sale_items`: بنود المبيعات
- `product_batches`: دفعات المنتجات
- `product_prices`: أسعار المنتجات
- `customer_credits`: الائتمانات
- `credit_payments`: دفعات الائتمان
- `app_settings`: إعدادات التطبيق
- `activity_log`: سجل الأنشطة
- `user_profiles`: ملفات المستخدمين

## المزامنة

- **تلقائية**: كل 5 دقائق عند الاتصال بالإنترنت
- **يدوية**: زر المزامنة في شريط التنقل العلوي
- **ذكية**: ترسل التغييرات المحلية وتحمل التحديثات من السحابة

## التطوير

### هيكل المشروع

```
src/
├── components/     # المكونات المشتركة
├── db/            # قاعدة البيانات والخدمات
│   ├── database.ts # إعداد Dexie
│   ├── services.ts # العمليات على البيانات
│   ├── sync.ts     # المزامنة مع Supabase
│   └── types.ts    # أنواع البيانات
├── layouts/       # تخطيطات الصفحات
├── lib/           # المساعدات والأدوات
├── pages/         # صفحات التطبيق
└── hooks/         # React Hooks مخصصة
```

### إضافة ميزة جديدة

1. أضف النوع في `types.ts`
2. أضف العمليات في `services.ts`
3. أضف المزامنة في `sync.ts`
4. أنشئ الصفحة في `pages/`
5. أضف الراوتر في `App.tsx`
